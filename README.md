# dev-config

## `start`

```sh
git clone https://github.com/rodionsibov/solid-octo-giggle dev-config
git clone  https://github.com/rodionsibov/urban-journey/tree/main cms
git clone https://github.com/rodionsibov/verbose-barnacle/tree/main frontend
cd cms && npm install
cd ../frontend && npm install
cd ../dev-config && docker compose up
```
